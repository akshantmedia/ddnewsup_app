# DD News UP App

DD News UP App is a digital news platform focused on delivering fast, reliable, and real-time news updates from Uttar Pradesh.

## 🚀 Features
- Latest News Updates
- Video News Content
- User-Friendly Interface
- Fast Loading Speed
- Mobile Optimized

## 📱 Purpose
This app is designed for news readers, journalists, and digital media users who want quick and accurate updates.
